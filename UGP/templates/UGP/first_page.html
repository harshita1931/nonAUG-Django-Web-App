What is non AUG?

Translation initiation using alternative codons is one of the important mechanisms by which single mRNA gives rise to multiple proteins. Several examples have been found in which mRNAs naturally support initiation from non-canonical codons. nonAUG database contains mRNA sequences with non-canonical start codons collected from RefSeq database. This database is the second version of the previously existing nonAUG database.
