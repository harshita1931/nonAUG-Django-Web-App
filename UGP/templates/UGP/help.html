Browsing nonAUG

  • You can browse through all the entries present in the nonAUG database through this option
  • Link to details of each protein also provided

Searching nonAUG

  • nonAUG can be searched by name of organism or by NCBI gene ID or by NCBI protein accession ID
  • The status of all the sequences present in this database is Reviewed

Sequence Details

Sequences present in the database have following details associated with them :
  • Name : Name of the protein
  • Protein Accession Number : NCBI protein accession ID
  • Organism name : Name of the organism in which the protein is present
  • Gene ID : ID of the gene from which the protein is made
  • mRNA Accession ID : NCBI Accession ID of the mRNA from which the protein is made
  • Status : Curation of the sequene
  • Start Codon : The start codon of the coding sequence for the protein
  • Upstream ATG count : Number of occurances of ATG in the upstream region of the coding sequence
  • Description : Description about the protein
  • Link to FASTA file : Downloadable link for the protein sequence id FASTA format

Statistics

  • Distribution of the organisms present in the database in terms of presence of non AUG codons is shown
  • Codon-wise classification for each non AUG codon for every organism present in database is shown
